DiffuMeta: Generative Framework for Metamaterial Design

generative modelingdiffusion transformersmetamaterials

Executive Summary

DiffuMeta is a generative framework that integrates diffusion transformers with algebraic language representation to design 3D shell structures. It aims to generate novel structures with targeted mechanical properties, addressing the one-to-many mapping challenge by producing diverse solutions.

Tech Stack / Tools

  • Diffusion transformers
  • Algebraic language representation

Current Status

Active development as of 2026-07-10, focusing on experimental validation of fabricated structures to confirm the efficacy of the approach.

Key Achievements & Functionality

  • Developed a compact parameterization for 3D geometries using mathematical sentences.
  • Enabled simultaneous control over multiple mechanical objectives, including linear and nonlinear responses.
  • Validated the approach through experimental tests on 3D-printed shell structures.
