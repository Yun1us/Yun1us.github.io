DiffuMeta: Generative Framework for Metamaterial Design

generative modelingmetamaterialsdiffusion models

Executive Summary

DiffuMeta is a generative framework that integrates diffusion transformers with an algebraic language representation to design 3D metamaterials. It encodes geometries as mathematical sentences, enabling the generation of shell structures with targeted stress-strain responses. The approach allows for simultaneous control over multiple mechanical objectives, addressing the one-to-many mapping challenge in inverse design.

Tech Stack / Tools

  • Diffusion Transformers
  • Algebraic Language Representation

Current Status

Active: The project is actively developing a framework for generating diverse shell structures with specific mechanical properties, validated through experimental tests on 3D-printed structures.

Key Achievements & Functionality

  • Developed a compact parameterization for 3D geometries using algebraic language.
  • Enabled generation of shell structures with targeted mechanical responses.
  • Addressed one-to-many mapping in inverse design by producing diverse solutions.
  • Validated model efficacy through experimental testing of fabricated structures.
