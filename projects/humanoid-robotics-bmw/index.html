Executive Summary

An interdisciplinary university project developed with BMW Group explored how humanoid robotics could support manufacturing tasks. The technical work focused on a VR-driven simulation architecture for intuitive robot teleoperation, demonstration capture, and a future imitation-learning workflow.

Tech Stack / Tools

  • ROS middleware
  • VR headset and controllers
  • Hand-tracking input
  • 3D robot simulation
  • Robot gripper and three-finger hand
  • Camera feedback pipeline
  • Imitation-learning workflow design

Current Status

Completed as a functional simulation and technical concept. The project delivered the core teleoperation, robot-control, visual-feedback, and setup infrastructure needed to demonstrate the approach, while identifying training-data scale and sim-to-real transfer as the next engineering challenges.

Key Achievements & Functionality

  • Designed a pipeline connecting VR input, teleoperation, ROS middleware, and a simulated humanoid robot.
  • Implemented control of the robot arms through VR controllers or hand recognition.
  • Integrated a simulated gripper and three-finger hand for manipulation experiments.
  • Returned the simulated camera view to the headset to close the operator feedback loop.
  • Consolidated startup and environment setup into a shared project structure for repeatable demonstrations.
  • Defined the path from recorded demonstrations to formatted training data and neural-network training.
  • Worked in a joint technical and business-studies team; the technical team owned the architecture, development environment, and simulation implementation while the business team translated the work into strategy and evaluation.
