Project

Neural Network Pruning for VGG Models on CIFAR-10

Completedneural networkspruningmachine learning

Executive Summary

This project explores fine-grained and channel pruning techniques applied to VGG models trained on the CIFAR-10 dataset. The goal is to reduce model size and improve efficiency while maintaining accuracy. The work demonstrates effective pruning strategies and their impact on model performance.

Tech Stack / Tools

  • PyTorch
  • NumPy
  • CIFAR-10 dataset
  • Matplotlib

Current Status

Completed: The project successfully implemented and evaluated pruning techniques, achieving high sparsity with minimal accuracy loss.

Key Achievements & Functionality

  • Implemented fine-grained pruning, achieving high sparsity with minimal accuracy loss.
  • Developed channel pruning methods, resulting in approximately 50% reduction in MACs and 20% latency improvement.
  • Compared channel sorting methods to preserve accuracy before fine-tuning.
