Technical case study

RODE: Learning Roles to Decompose Multi-Agent Tasks

Multi-Agent Reinforcement LearningRole DecompositionArtificial Intelligence

Executive Summary

RODE addresses the challenge of scaling Multi-Agent Reinforcement Learning (MARL) by decomposing tasks into role-specific action spaces, reducing the combinatorial explosion of joint actions. This approach enhances coordination among agents in complex environments, such as the StarCraft Multi-Agent Challenge (SMAC), by using a bi-level hierarchy of role selection and execution.

Tech Stack / Tools

  • Reinforcement Learning
  • k-means Clustering
  • QMIX-style Mixing Networks
  • StarCraft Multi-Agent Challenge (SMAC) for validation

Current Status

Completed: The project was presented at ICLR 2021, demonstrating state-of-the-art performance in multi-agent coordination tasks.

Key Achievements & Functionality

  • Developed a framework that learns action-effect embeddings to define roles automatically.
  • Implemented a bi-level hierarchy for role selection and execution, improving sample efficiency.
  • Achieved state-of-the-art results on 14 SMAC maps, particularly excelling in super-hard scenarios.
  • Demonstrated zero-shot transfer capabilities, maintaining performance in novel environments without retraining.
