Quantization Techniques for Neural Networks

quantizationneural networksdeep learning

Executive Summary

This project explores various quantization techniques to reduce model size and improve inference speed for neural networks. It focuses on implementing k-means and linear quantization methods to compress a VGG model trained on CIFAR10.

Tech Stack / Tools

  • Python 3.11
  • PyTorch
  • fast-pytorch-kmeans
  • NumPy
  • Matplotlib
  • Torchvision

Current Status

Active: The project is actively exploring quantization methods, with implementations for k-means and linear quantization completed.

Key Achievements & Functionality

  • Implemented k-means quantization using fast-pytorch-kmeans to cluster synapses and reduce model size.
  • Developed linear quantization methods to directly round floating-point values to quantized integers.
  • Achieved a reduction in model size and latency while maintaining acceptable accuracy levels.
  • Provided visualization tools to analyze the distribution of quantized weights and validate quantization methods.
