  2. 02ML × Systems
    Completed2026

    Neural Network Pruning for VGG Models on CIFAR-10

    This project explores fine-grained and channel pruning techniques applied to VGG models trained on the CIFAR-10 dataset. The goal is to reduce model size and improve efficiency while maintaining accuracy. The work demonstrates effective pruning strategies and their impact on model performance.

    neural networkspruningmachine learning
    Open project ↗
  3. 03Generative × Engineering
    Active2026

    DiffuMeta: Generative Framework for Metamaterial Design

    DiffuMeta is a generative framework that integrates diffusion transformers with algebraic language representation to design 3D shell structures. It aims to generate novel structures with targeted mechanical properties, addressing the one-to-many mapping challenge by producing diverse solutions.

    generative modelingdiffusion transformersmetamaterials
    Open project ↗
  4. 04Efficiency × Inference
    Active2026

    Quantization Techniques for Neural Networks

    This project explores various quantization techniques to reduce model size and improve inference speed for neural networks. It focuses on implementing k-means and linear quantization methods to compress a VGG model trained on CIFAR10.

    quantizationneural networksdeep learning
    Open project ↗
  5. 05MARL × Research
    Completed2026

    Seminar Presentation: RODE and Role-Based Multi-Agent Learning

    This seminar presentation analyzed and explained RODE: Learning Roles to Decompose Multi-Agent Tasks, an existing research paper presented at ICLR 2021. The presentation focused on how the paper decomposes Multi-Agent Reinforcement Learning tasks into role-specific action spaces and uses a bi-level hierarchy to improve coordination; it was not my own paper or original implementation.
